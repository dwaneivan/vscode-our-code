## Our Code

Simply _**Annoy**_ Other Coders.

## Usage

Press `F1` or `CMD/CTRL + Shift + P` and Run the Command Name

**List of Command Names**

- Our Code: Space
- Our Code: Semicolon
- Our Code: Space Mayhem
- Our Code: Semicolon Mayhem
